# Agent instructions

GratiText helps people send thoughtful gratitude notes to loved ones on a
regular schedule.

## Essentials

- Node version: `^24`
- Build: `bun run build` (runs `build:icons`, then `react-router build`)
- Typecheck: `bun run typecheck` (runs `react-router typegen`, then `tsc`)
